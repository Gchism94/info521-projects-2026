# Course-document PDFs

UA-styled PDFs (Typst; same look as the hub syllabus) of the checkpoint slips,
project overviews, and grading reference — for D2L upload. Each carries a
"Course-schedule placement" cover block mapping it to the week/module arc
(the syllabus §1.6 pacing).

⚠ The five plain checkpoint PDFs include the **answer keys** — instructor sitting
copies only. The `*-STUDENT.pdf` variants are the D2L-safe versions: prompts
(Versions A + B) and Satisfactory criteria, **no keys** (verified key-free at
build time). Post only the STUDENT versions.

Regenerate after editing any source doc (the placement text lives in the build
script, not the sources).
