# Data

`clinical_reference.csv` — **synthetic** reference dataset (400 patients, 7
features + continuous `risk` target). Generated by `generate_reference_data.py`
(seed 521). It exists so the pipeline runs reproducibly out of the box.

**This is not real clinical data.** For deployment, swap in a curated real dataset
— see `../instructor/dataset-curation.md`.

Regenerate:

```bash
python generate_reference_data.py
```

Column contract honored by `info521.data.load_clinical`: header row; continuous
target column named `risk`; an `age` column serving as the Project 1 primary
predictor.
