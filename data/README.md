# Data

## `nhanes_2021_2022.csv` — deployed default (real data)

The dataset `info521.data.load_clinical()` loads by default. A curated extract of
**NHANES 2021–2022** (CDC, cycle suffix `_L`): 5,102 adults aged 18–79, complete
cases on all columns. Built by `build_nhanes.py`, which downloads six public
NHANES XPT files, merges them on `SEQN`, and derives oscillometric blood pressure.

Columns, in order: `age, bmi, waist, chol, hdl, hba1c, sbp, dbp`.

- **Target** (`sbp`): systolic blood pressure (mmHg), the continuous Project 1
  outcome.
- **Reserved** (`dbp`): diastolic blood pressure. Used *only* to build the
  Project 2 hypertension label (ACC/AHA: SBP ≥ 130 or DBP ≥ 80) via
  `info521.data.hypertension(ds)`. It is excluded from the feature matrix and is
  never the target, so the hypertension label can't leak into its own predictors.
- **Features**: `age, bmi, waist, chol, hdl, hba1c`.

Survey weights are deliberately ignored — this is an algorithms course about
estimators, not population inference.

Rebuild (downloads are cached in the gitignored `raw/`):

```bash
python build_nhanes.py
```

## `clinical_reference.csv` — synthetic offline fallback

A **synthetic** reference dataset (400 patients) generated by
`generate_reference_data.py` (seed 521). It exists so the pipeline runs
reproducibly with no network access. **This is not real clinical data.** Load it
explicitly when offline:

```python
info521.data.load_clinical(path="data/clinical_reference.csv")
```

Regenerate:

```bash
python generate_reference_data.py
```

## Column contract (honored by `info521.data.load_clinical`)

Header row; a continuous target column named `sbp`; a `dbp` column, if present, is
reserved (excluded from features and surfaced separately for the hypertension
label); an `age` column serving as the Project 1 primary predictor. See
`../instructor/dataset-curation.md`.
